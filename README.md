# imageholics

## Install & Update

Install the following browser extensions accordingly:

   * Chrome: TamperMonkey
   * Safari: NinjaKit
   * Firefox: GreaseMonkey

Visit [the script](https://raw.githubusercontent.com/initiumlab/imageholics/master/Imageholics.user.js) and accept to install.
